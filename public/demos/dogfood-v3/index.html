Stop shipping AI slop.

Design Guard is a design intelligence CLI that lints AI-generated HTML against your design system. It catches off-palette colors, default fonts, heading hierarchy issues, and accessibility gaps before they reach production.

What Design Guard Catches

Eight modular lint rules detect the patterns that make AI-generated sites look like AI-generated sites.

Anti-Slop Validation

Scores HTML 0-100 against your DESIGN.md. Catches off-palette colors, AI-default fonts like Inter and Poppins, purple-to-blue gradients, uniform icon grids, div soup, duplicate CTAs, heading hierarchy skips, and missing accessibility attributes. Every rule maps to a specific symptom of AI-generated design.

Design Intelligence Agent

Before generating a single pixel, the dg discover command researches your business model, scrapes your existing site, analyzes 2-3 competitors, and scores its own confidence. It understands that a grocery chain needs a store locator, not a shopping cart.

Generates an 8-section DESIGN.md with business context baked in: visual theme, color palette, typography, spacing, components, iconography, imagery, and do's/don'ts.

Multi-Generator Support

Works with Google Stitch MCP and Claude Direct. Generate screens through either pipeline and lint the output the same way. Not locked to one AI vendor.

Three Output Frameworks

Export to Static HTML, Astro, or Next.js App Router. Each build includes a Design Guard signature comment and shared navigation across pages.

W3C DTCG Token Bridge

Import and export design tokens in the W3C Design Tokens Community Group format. Bridge your DESIGN.md to standard tooling or pull tokens from your existing design system into Design Guard.

$ dg tokens --export tokens.json
# Exports DESIGN.md as W3C DTCG tokens

13 Commands. 7 Skills.

Everything from research to deployment, accessible from the terminal or from Claude Code as slash commands.

CLI Commands

  • dg init Setup project, auth, and MCP config
  • dg discover Research business + generate tailored DESIGN.md
  • dg design Generate DESIGN.md from a brand brief
  • dg generate Create a screen via Stitch or Claude
  • dg lint Score HTML against your design system
  • dg build Export to Static, Astro, or Next.js
  • dg preview Open screens in the browser
  • dg sync Pull screens from a Stitch project
  • dg research Check for Stitch API updates
  • dg workflow Guided multi-step workflows
  • dg quota View generation quota usage
  • dg tokens W3C DTCG token import/export
  • dg tui Launch interactive terminal UI

Start Linting in 30 Seconds

Install globally, point it at your HTML, and get a score. No config needed for basic linting.

npm i -g design-guard
or use the core library in your own tools
npm i @design-guard/core
